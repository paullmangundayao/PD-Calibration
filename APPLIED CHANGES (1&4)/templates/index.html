Activate Initial Sealing Button: Activates the initial sealing process so that the base of the bubble wrap is already wrapped before proceeding with the delivery of the product.

Detect Dimensions Button: Captures object dimensions, displays the image with a bounding box in the Captured Image container and shows the detected measurements in the Detected Dimensions container.

Clear Results Button: Clears the captured image, detected dimensions, and results.

Emergency Stop Button: Stops the current process happening.

About the Project:

This project, named EXACTOPACK, is a design of packaging material optimization system for an automated bubble wrap packaging process, focusing on enhancing efficiency and reducing plastic waste in e-commerce. The system automates the measurement of products using web cameras, optimizes the use of bubble wrap using a metaheuristic algorithm, automates the wrapping of the product using the optimized bubble wrap size, and delivers fully packaged products ready for final labeling and delivery.

Zero Step: Set-up the bubble wrap first, then click "activate initial sealing" button.
First Step: Place the product inside, turn on the lights using a remote, click "detect dimensions" on the web application.
Second Step: Verify the results, wait for the modal to popup, choose if you will deliver the item or clear the results to restart.
Third Step: Wait for the product to slide and check its position on the delivery platform.
Fourth Step: Wait for it to be delivered inside the bubble wrap. A modal will popup again, choose if you want to proceed with the sealing or not.
Fifth Step: Wait for the finished product at the bottom of the black table.
